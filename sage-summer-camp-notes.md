What is Sage? Project that has been happening for about 10 years, built off of "array of things", 25 Million Dollar project

- Need to deploy 300 new state-of-the art AI-enabled platform
- A testbed 
- Why process on the edge? Badwidth, privacy, transmitting takes a lot of power, latancy, resiliency (you want each system to be atonymous on it's own, if you pull the cable on a node, then the work keeps going)
- Real time hazard detection
- Infrared Camera in Oregon - pretty cool, I'm from there, so close to home
